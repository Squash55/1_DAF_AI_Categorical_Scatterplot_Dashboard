
# Air Force AI Dashboard

## Features
- Jittered scatterplot with background zones (Blue/Red)
- Rule-based and GPT-based insight generation
- Dynamic auto-update for new uploads
- AI-generated data disclaimer with confidence levels
- Designed for strategic cyber risk intelligence

## Setup
1. Install dependencies:
   pip install -r requirements.txt

2. Add a `.env` file in this folder with your OpenAI key:
   OPENAI_API_KEY=sk-...

3. Run:
   streamlit run airforce_dashboard.py
